### Environment Variables

This project uses environment variables for the backend server.  
They are required for the AI feature to work.

Create a `.env` file inside the **server** folder and add these values:

```env
PORT=3000
OPENROUTER_API_KEY=your_openrouter_api_key
OPENROUTER_MODEL=your_model_name

```

### Starting the Application (`start.bat`)

The project includes a `start.bat` file to make running the app easy on Windows.

After cloning the repository and setting the `.env` file, simply run:

```bash
start.bat
